# Code of Vitiligo Lesion Segmentation  
The code generates segmentation labels of an input image.
![DEMO](https://github.com/YuhanZheng0327/Weakly-Supervised-Vitiligo-Lesion-Segmentation/blob/master/GITHUB.jpg)

python 3.9  
pytorch 2.3.1

numpy 1.24.3  
opencv2
